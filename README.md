# MULAN-SKILLS

一个用于沉淀与分享「技能说明与可复用工作流」的仓库。每个技能以独立目录维护，包含任务目标、操作说明、提示词模板与产出格式，便于快速复用与协作。

## 目录结构
- `video-analyst/`：视频分析技能说明与工作流
  - 详见 [SKILL.md](file:///d:/Projects/Gemini/Github项目管理/MULAN-SKILLS/video-analyst/SKILL.md)

## 使用方式
- 直接阅读各技能目录下的 `SKILL.md`，按步骤执行或复制提示词到你的 AI 助手。
- 如需新增技能：在仓库根目录创建同名文件夹，并编写该技能的 `SKILL.md`。

## 快速开发与推送
1. 在本地修改或新增技能文档
2. 执行：
   - `git add .`
   - `git commit -m "Update skills: <your-change>"`
   - `git push`
